# H2E_kotlin
- KSB 공모전을 위해 제작한 첫번째 앱

- 기능

1. firebase-firestore를 이용한 서버연동 (로그인, 데이터 입,출력)
2. 리사이클러뷰
3. 안드로이드 스튜디오 내장 stt 라이브러리를 이용한 stt기능 삽입
